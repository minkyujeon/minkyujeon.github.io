Autoresearch v2 demo: existing results vs baseline

Presentation snapshot using preserved completed runs. Denominators are shown explicitly, so N=32 and N=96 results can be compared by success percentage without pretending they are identical experiments.

Core message

On several hard targets, the v2 search finds configurations that beat the local default/fixed baseline in raw success rate and structure quality metrics. CD45 is the clearest demo case: baseline N=32 is 8/32, ours best available audited tree node is 22/32 with 5 unique clusters, a larger search node reaches 86/192, and the fixed rerun reaches 41/96.

Success criterion follows the existing local analysis: iPAE pass, pLDDT pass, and final binder success. Lower iPAE is better; higher pLDDT is better.
