Definitions

Strict Global Unique

Sequences are deduplicated across the whole run before counting. A repeated successful sequence from a later node does not increase the cumulative count.

Baseline

Official-style seeded baseline runs repeat the published beam-search style configuration with changing random seeds on one GPU per target.

Adaptive Runs

Adaptive runs build a search tree over candidate generation settings and use measured outcomes from previous nodes to decide the next nodes.
