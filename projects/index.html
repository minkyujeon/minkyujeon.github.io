Token Ordering

Learned Token Ordering for Visual Autoregressive Generation

Investigating whether a learned generation order (via OrderHead) can improve image quality in autoregressive models. Systematic evaluation across MAR (masked) and LlamaGen (causal) architectures, with 50+ experiments covering heuristic orderings, STE-based learned orderings, oracle supervision, and random-order training (RAR).

Inference-Time Scaling

Agentic Inference-Time Scaling for De Novo Protein Design

Mapping the full action space of a generative protein design pipeline — search algorithms, reward composition, denoising temperature, sequence refinement — and building an agent that adaptively routes compute to achieve scaling that doesn't plateau. Includes an audited results map, an open research plan, and a companion implementation-notes page covering the markdown design history, current code path, and Gemma/Qwen backend experiments.
