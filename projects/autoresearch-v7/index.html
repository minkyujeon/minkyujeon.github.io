How to read this page

The endpoint hierarchy mirrors the paper: generation throughput first, independent quality second, candidate nomination last.

Primary endpoint

Strict successpLDDT ≥ 90, iPAE ≤ 7/31 (0.2258), and binder scRMSD < 1.5 Å.
Structure-unique (SU)Strict successes clustered by Foldseek on the binder chain at TM0.6.
Unique sequencesAll strict successes clustered with MMseqs2 at the sequence-identity threshold; this does not replace SU.
Independent scoresAF2 ipTM/ipSAE/actifpTM and other metrics are post-hoc diagnostics, not SU credit.

Fair-comparison rules

Method-only baselinesComplexa, BindCraft and BoltzGen use their official default configurations.
Controller baselinesPUCT and ε-greedy use the same registered action space and execution stack as T-ReX; only proposal/allocation logic differs.
Compute accountingRates use worker GPU-hours. T-ReX LLM/controller GPU time is not added to the worker denominator.
Top-K interpretationTop-10/50/100 panels are selected after strict filtering and structure clustering; threshold counts use the full scored SU pool.
