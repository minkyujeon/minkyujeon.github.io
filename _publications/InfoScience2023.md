Youngjin Oh\*, Minkyu Jeon\*, Dohwan Ko, Hyunwoo J Kim

Many self-supervised representation learning methods have achieved high performance in
image classification tasks. However, these methods have limited performance on localization tasks such as object detection or semantic segmentation. Most self-supervised representation learning methods are optimized with only one global representation, which does
not pay much attention to the spatial information in an image. We propose a simple and
effective method that uses the positional relationships between the entities in an image
by shuffling the convolution kernels. Our method extends current self-supervised learning
and calculates the pixel-wise (dis) similarities between the output of the standard convolution kernels and that of the randomly shuffled convolution kernels. Our proposed method
achieves higher performance on object detection, instance segmentation, and semantic
segmentation when attached to recent self-supervised learning methods.
